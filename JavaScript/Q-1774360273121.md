# About `Infinity` & `-Infinity`

`Infinity` & `-Infinity` are special numeric values.  

`Infinity` represents the mathematical Infinity ∞.  
It is a special value that’s greater than any number.  

We get `Infinity` when a positive number is divided by zero.  

We get `-Infinity` when a negative number is divided by zero.  

> `0 / 0` gives `NaN`, not `Infinity` or `-Infinity`

```javascript
console.log(5/0); // Infinity
console.log(-5/0); // -Infinity
console.log(0/0); // NaN

console.log(Infinity + 5); // Infinity
console.log(Infinity - 5); // Infinity
console.log(Infinity / 5); // Infinity
console.log(Infinity * 5); // Infinity

console.log(-Infinity * -1);  // Infinity  ← negatives cancel out, standard math
console.log(-Infinity * -5);  // Infinity  ← negatives cancel out, standard math
console.log(-Infinity / 5);   // -Infinity

console.log(Infinity + Infinity); // Infinity
console.log(Infinity - Infinity); // NaN
console.log(-Infinity + Infinity); // NaN
console.log(-Infinity - (-Infinity)); // NaN

console.log(Infinity + NaN); // NaN
console.log(Infinity - NaN); // NaN
console.log(Infinity / NaN); // NaN
console.log(Infinity * NaN); // NaN

// isFinite() — a useful built-in
console.log(isFinite(100));       // true
console.log(isFinite(Infinity));  // false
console.log(isFinite(-Infinity)); // false
console.log(isFinite(NaN));       // false  ← also catches NaN
```

---

## Quick Mental Model

| Operation | Result |
|---|---|
| positive ÷ 0 | `Infinity` |
| negative ÷ 0 | `-Infinity` |
| 0 ÷ 0 | `NaN` |
| `Infinity - Infinity` | `NaN` |
| anything with `NaN` | `NaN` |
