# Coding Question

What will be the output of the following:  

```javascript
const obj = {
    a: 1,
    b: () => console.log(this.a),
    c: function () { console.log(this.a) }
}

console.log(obj.a) // 1
obj.b() // undefined
obj.c() // 1
```

Output:  
1  
undefined  
1  

Explanation:  

```javascript
const obj = {
    a: 1,
    b: () => console.log(this.a), // Here this points to Global this/window object (immediately surrounding lexical scope is Global execution context), "obj" is in the Global execution context.
    c: function () { console.log(this.a) } // Here due to "Implicit Binding" this points to the object that calls the method.
}
```
