# Get count for the each character in the string 'Times of India'

```javascript
const toi = 'Times of India';

const characterCounts = new Map();

for (char of toi.toLowerCase()) {
    let count = characterCounts.get(char);
    count = !count ? 1 : (count + 1);
    characterCounts.set(char, count);
}

console.log(characterCounts);
```
