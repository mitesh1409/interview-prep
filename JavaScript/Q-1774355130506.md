# Coding Question

Source: https://www.youtube.com/shorts/wDuJUiJfv4k  

Given a string, write code to find out count of each character.  

---

## Solution

Solution #1: Using Array and forEach

```javascript
const krishnaJap = 'hare krishna hare krishna krishna krishna hare hare';

const characters = [];

[...krishnaJap].forEach((character) => {
    if (!characters[character]) {
        characters[character] = 1;
        return;
    }
    characters[character]++;
});

// Characters with their count.
console.log(characters);
```

Solution #2: Using Object and for...of

```javascript
const krishnaJap = 'hare krishna hare krishna krishna krishna hare hare';

const characters = {};

for (let character of krishnaJap) {
    if (!characters[character]) {
        characters[character] = 1;
        continue;
    }
    characters[character]++;
}

// Characters with their count.
console.log(characters);
```

Solution #3: Using map and for...of

```javascript
const toi = 'Times of India';

function getCharacterCounts(str, caseSensitive = true) {
    const characterCounts = new Map();

    if (!caseSensitive) {
        str = str.toLowerCase();
    }

    for (char of str) {
        let count = characterCounts.get(char);
        count = !count ? 1 : (count + 1);
        characterCounts.set(char, count);
    }

    return characterCounts;
}

console.log(getCharacterCounts(toi));
```
