# Coding Question

Write a function to check if a number is an integer without using any standard/built-in function.

```javascript
const isInt = (number) => (number * 10) % 10 === 0;

// Tests
console.log(isInt(3.14)); // false
console.log(isInt(314)); // true
console.log(isInt(1.1)); // false
console.log(isInt(100)); // true
console.log(isInt(99.99)); // false
console.log(isInt(99)); // true
```
