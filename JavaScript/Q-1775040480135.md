# Coding Question

Empty JavaScript array in one line of code.

```js
let numbers = [1, 2, 3, 4, 5];
```

There are a few ways to empty a JavaScript array in one line:

**1. Set length to 0** *(best for mutating in place, keeps original reference and mutates original array)*

```js
numbers.length = 0;
```

**2. Reassign to empty array** *(simple, but loses original reference)*

```js
numbers = [];  // requires `let`, not `const`
```

**3. splice()**  *(mutates original)*

```js
numbers.splice(0, numbers.length);
```

**4. pop() in a loop (one-liner)** *(less common)*

```js
while (numbers.pop()) {}
```

---

**Recommendation:** Use `numbers.length = 0` — it mutates the original array in place, so all references to it stay in sync, and it works with `const`.
