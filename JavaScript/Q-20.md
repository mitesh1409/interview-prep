```JavaScript
// Objects in JavaScript
// What is the output of the following code?

const a = {};
const b = { key: 'b' };
const c = { key: 'c' };

a[b] = 123;
a[c] = 456;

console.log(a[b]);

// Output:
// 456

// Explanation:
// In JavaScript, when you use an object (like 'b' or 'c') as a key in another object (like 'a'),
// it gets converted to a string. Both 'b' and 'c' are converted to the same string "[object Object]".
// Therefore, when you assign a[c] = 456, it overwrites the previous value of a[b] = 123.
// As a result, a[b] returns 456.

// To avoid this issue, you can use a Map instead of a plain object for key-value pairs
// where the keys are objects.

const map = new Map();
map.set(b, 123);
map.set(c, 456);

console.log(map.get(b));

// Output:
// 123

// Explanation:
// In a Map, the keys can be objects and they are not converted to strings.
// Therefore, map.get(b) correctly returns 123.
```
