# About `NaN`

`NaN` = Not a Number  
`NaN` is a special numeric value.  

`NaN` represents a computational error.  
It is a result of an incorrect or an undefined mathematical operation.  

```javascript
// Trying to cut an apple into 2 pieces
const halfApple = 'Apple'/2;

console.log('halfApple', halfApple);
// Output
// halfApple NaN
```

`NaN` is sticky. Any further mathematical operation on `NaN` returns `NaN`.  

```javascript
console.log( NaN + 1 ); // NaN
console.log( 3 * NaN ); // NaN
console.log( NaN / 5 ); // NaN
```

If there’s a `NaN` somewhere in a mathematical expression, it propagates to the whole result (there’s only one exception to that: `NaN ** 0` is 1).

```javascript
console.log(NaN ** 0);
```
