# What is the difference between Runtime Environment and Framework?

**Runtime Environment**  
It provides the necessary infrastructure for "code execution",  
including services like memory management, I/O operations etc.

Some examples are:  

| Programming Language | Runtime Environment |
| :--- | :--- |
| JavaScript | Node.js |
| Java | JVM |
| Python | CPython |

**Framework**  
It offers structured set of tools, libraries and best practices  
to simplify the development process.

Some examples are:

| Programming Language | Runtime Environment |
| :--- | :--- |
| JavaScript | [Angular](https://angular.dev/), [NestJS](https://nestjs.com/) |
| Java | [spring](https://spring.io/) |
| Python | [django](https://www.djangoproject.com/) |
