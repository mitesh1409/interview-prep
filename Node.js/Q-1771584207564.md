# What is the difference between Node.js and Express.js?

Node.js is a "JavaScript Runtime Environment" that allows execution of JavaScript code on the Server-Side (outside of a Browser).

Express.js is a "Framework" built on top of Node.js.

It provides features like:  

- Routing
- Middleware
- Template Engines
- Database Integration
- Error Handling

Refer Express documentation to get more details - [express](https://expressjs.com/).
