# How would you handle multiple asynchronous calls efficiently?

Promise.all executes multiple promises in parallel, improving performance.

```javascript
const fetchUsers = fetch("https://api.example.com/users");
const fetchOrders = fetch("https://api.example.com/orders");

Promise.all([fetchUsers, fetchOrders])
    .then(results => Promise.all(results.map(res => res.json())))
    .then(data => console.log(data))
    .catch(error => console.error(error));
```
