# What are Modules in Node.js? Module vs Function

A module contains a specific functionality.  
It can have one or more functions to achieve that functionality.  
Each module has its own seperate file.  
We can import module to use its functionality wherever it is required.  

A function has a set of instructions to do a particular job/task.  

Example  

```JavaScript
// Loan.js module file.

function calculateEmi(loanAmount, tenure) {
    return parseInt(loanAmount/tenure);
}

function getEmis(loanAmount, tenure) {
    const emiAmount = parseInt(calculateEmi(loanAmount, tenure));
    const emis = [];
    for (let i = 0; i <= tenure; i++) {
        emis.push(emiAmount);
    }

    const adjustment = loanAmount - (emiAmount * tenure);
    // Adjustment added in the first emi.
    emis[0] += adjustment;

    return emis;
}

function checkEligibility(loanAmount, creditScore) {
    // Checks eligibility as per the credit score.
    // Returns true/false.
    return true;
}

export {
    calculateEmi,
    getEmis,
    checkEligibility
};

// app.js file.

// For example we are using Loan.js module in our app.js file.
import { calculateEmi, getEmis, checkEligibility } from './Loan.js';

// ...

```
