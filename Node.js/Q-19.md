# What are different types of modules in Node.js?

There are 3 types of modules in Node.js:  

## #1 Core/Built-in Modules

These modules are provided by Node.js out of the box, they are built and managed by Node.js team.  
Examples are - fs, path, http, https, util, events, console, cluster etc.

## #2 Custom/local Modules

These modules are built by developers for thier application.

## #3 Third Party Modules

These are provided by third party packages that we install in our application as per the requirement.  
They are built and managed by third party.
